Acerca de:

Este proyecto fue desarrollado como parte de un proceso de aprendizaje autodidacta con el objetivo de practicar y aplicar conocimientos fundamentales de desarrollo web, utilizando tecnologías como HTML, CSS y JavaScript. El resultado es una pequeña página web interactiva que incluye tanto un videojuego estilo arcade como secciones informativas, como esta misma página “Acerca de”, diseñadas para simular un proyecto real de sitio web completo. El juego, inspirado en los clásicos de plataformas, tiene como objetivo brindar una experiencia entretenida en la que el jugador debe esquivar obstáculos y recolectar puntos, controlando un personaje simpático que se mueve y salta con fluidez. Además del juego, se desarrollaron elementos comunes en sitios web, como diseño responsivo, botones interactivos, estructura clara, y la integración de estilos visuales agradables que mejoran la experiencia del usuario.
